=== About ===
This plugin allows you to change the default sender name and sender email of your WordPress website to a more personalised value. Make your website visitors know it is you sending the emails.

=== Installation Instructions ===
1. Upload `change-sender-id` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the 'Change Sender ID' link from Settings on the main menu
4. Fill in the 'Sender Name'
5. Fill in the 'Sender Email Address'
6. Click the 'Save' button
